# 🕯️ התראת שבת — Shabbat Alert

אפליקציית **iPhone / Android** שמתריעה אוטומטית על כניסת שבת — בלי לכוון שעון, בלי לעשות כלום.

## מה האפליקציה עושה?

| # | פעולה | פירוט |
|---|-------|-------|
| 1 | **מתעוררת רק בימי שישי** | משימת רקע שרצה כל 6 שעות, פועלת רק ביום חמישי/שישי |
| 2 | **בודקת שעת כניסת שבת** | חישוב אסטרונומי מדויק של השקיעה לפי מיקום GPS |
| 3 | **מזהה מיקום אוטומטית** | לפי ה-GPS של הפלאפון — עובד בכל מקום בעולם |
| 4 | **התראה ראשונה — שעה לפני** | 🔔 צליל + הודעה "שבת בעוד שעה!" |
| 5 | **התראה שנייה — בזמן כניסת שבת** | 🕯️ צליל + הודעה "שבת שלום! הגיע זמן הדלקת נרות" |

## טכנולוגיה

- **React Native (Expo)** — קוד אחד ל-iOS ו-Android
- **expo-location** — שירותי מיקום
- **expo-notifications** — התראות מקומיות
- **expo-background-fetch** — משימות רקע
- **חישוב אסטרונומי** — נוסחאות NOAA לחישוב שקיעה (ללא תלות ב-API חיצוני)

## הפעלה

```bash
# התקנת תלויות
cd shabbat-alert
npm install

# הפעלה במצב פיתוח
npx expo start

# הפעלה ישירות על מכשיר
npx expo start --android
npx expo start --ios
```

## בנייה להפצה

```bash
# בניית APK לאנדרואיד
npx expo build:android

# בניית IPA לאייפון
npx expo build:ios

# או באמצעות EAS Build
npx eas build --platform android
npx eas build --platform ios
```

## מבנה הפרויקט

```
shabbat-alert/
├── App.js                  # מסך ראשי עם ממשק בעברית
├── src/
│   ├── shabbatTimes.js     # חישוב אסטרונומי של זמני שבת
│   ├── notifications.js    # ניהול התראות מקומיות
│   ├── location.js         # שירותי מיקום GPS
│   └── backgroundTask.js   # משימת רקע אוטומטית

└── assets/                 # אייקונים ותמונות
```

## הרשאות נדרשות

- **מיקום** — לחישוב זמני שבת לפי מיקום נוכחי
- **התראות** — לשליחת התראות על כניסת שבת
- **רקע** — להפעלת חישוב אוטומטי ללא פתיחת האפליקציה
